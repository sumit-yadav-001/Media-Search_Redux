🎬 Redux Media Search App

A powerful Redux-based Media Search Application built with React and Redux.
This app fetches and displays images, videos, and GIFs from an external API using global state management and efficient UI rendering.

🔗 Live Demo

👉 https://media-search-redux-project.netlify.app/

📌 Features

✅ Global state management with Redux

✅ Search functionality for images, videos & GIFs

✅ API integration for rich media results

✅ Responsive UI with clean layout

✅ Loading & error handling

✅ Modular and reusable components

JavaScript (ES6+)

CSS3
